Country-wide Shipping Optimization

Project Overview

Optimized the multi-product shipment strategy across a national network to minimize costs and meet demand, considering constraints like inventory levels, truck availability, labor hours, and node capacities.

Features

  • ⚡ Quick scenario simulations: Dynamic strategy testing
  • 🎯 Objective: Fulfill the demand, maximize the network resilience, minimize costs

Workforce Assignment Optimization

Project Overview

Multi-objective optimization algorithm that matches clients with salespeople based on skills, capacity, revenue potential, and efficiency scores in real-time.

Features

  • ⚡ Real-time matching: Dynamic assignment visualization
  • 🎯 Multi-objective: Balances satisfaction, revenue, and workload
  • 🔄 Adaptive: Handles urgent clients and capacity changes
  • 📊 Scoring system: Weighted compatibility matrix

Algorithm Insights

The algorithm evaluates skill compatibility, revenue potential, and workload distribution to create optimal assignments. Random events simulate real-world dynamics like urgent requests and capacity changes.
